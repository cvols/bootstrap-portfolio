About Me

My name is Chris Volonnino and I'm currently located in Hoboken, NJ.

I've been developing my own websites for over the past year or so, and just recently enrolled in Rutgers Full Stack Coding Bootcamp. I've deployed one web site with one of my friends who's father works for a non profit in Edison NJ. I developed the entire front end from scratch, using HTML5, CSS3, Bootstrap, and Angular 2.

I'm looking to obtain a full-time frontend/UI web developer position where I can apply my interpersonal skills paired with my developer experience to help make strategic impacts for a company while solving functional and web design problems.

What I Know:

  • Creating a webpage from scratch: HTML5, CSS3, Bootstrap, GitHub, and Git Version Control
  • Creating responsive websites using media queries
  • Mastering terminal commands
  • JavaScript and it's most beloved child, jQuery
  • Consuming RESTful API's
  • Parsing JSON to extract meaningful data
  • Using AJAX to update data on a website without having to hit the "refresh" button in the browser
  • Writing Node.js server code to serve static web pages

What I will learn from Rutgers Bootcamp:

  • Querying large amounts of data and answering questions from a MySQL Database
  • Understanding and using Joins, Wheres, and Counts strategically
  • Create scalable web apps, API's, and Services
  • Take a deep dive into core Java and Object Oriented Programming
  • Build a foundation in common build tools for Java projects, such as Maven
  • Computer Science applied to JavaScript
  • Design Patterns
  • Algorithms

Why coding? I love problem solving and the ability to be creative. Coding gives me the opportunity to work with team members and constantly involve by changing and learning. I love the ability to constantly grow and will do whatever it takes to be the best.
